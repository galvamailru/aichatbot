Сервер с функционалом AI агента в виде чата, который можно встраивать в html через <iframe>. Код реализации Python, docker. Быстрый запуск через docker-compose. AI агент использует LLM deepseak. Должна быть возможность изменнения ссылки на LLM сервер. Все переменные окружения должны быть в файле .env.  Промпт LLM должен читаться из отдельного файла. Диалоги пользователя должны быть разделены по USER_ID и сохранятся в БД PostgresSQL.
